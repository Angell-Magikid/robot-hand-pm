## Project Overview

面向生物实验室自动化场景的机器人解决方案，通过灵巧手、机械臂与智能控制系统，提升实验室操作效率与标准化水平。

Focus: Lab pipetting, grasping, squeezing, and transfer automation using dexterous robotic grippers.
